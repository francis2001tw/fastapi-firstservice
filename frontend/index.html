用户认证演示

FastAPI + Supabase · 邮箱 / 手机 / Google / 微信

邮箱模式区分注册与登录: 新用户请先注册, 已有账号直接登录。
手机 OTP 不区分注册/登录: 如果手机号未注册, Supabase 会自动创建账号。
需在 Supabase Dashboard 启用 Phone Provider 并配置 Twilio。
Google OAuth 不区分注册/登录: 首次使用 Google 登录会自动注册。
使用 PKCE 流程, 后端处理 callback。
微信登录由后端独立实现 (Supabase 不原生支持)。
当前为 Mock 模式: 点击登录将使用模拟微信身份。
真实模式需配置微信开放平台 AppID/Secret。
